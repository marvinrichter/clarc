What is clarc

Not an assistant.
An engineering system.

clarc installs into ~/.claude/ and wires 61 specialized agents, 247 domain skills, and 172 slash commands into a structured, opinionated workflow.

→
Agents route automatically
Write Go code → go-reviewer invoked. Fail a build → build-error-resolver. No config.
→
Skills load on demand
247 deep workflow patterns. Only loaded when relevant. Zero context overhead.
→
Gets better with use
Continuous learning flywheel extracts patterns from every session.

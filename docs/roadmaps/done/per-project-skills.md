# Project-Local Skills

**Status:** ✅ Done | **Date:** 2026-03-09 | **PR:** #12

**Goal:** Formal mechanism for project-specific skills that don't pollute global `~/.clarc/`.

## Key Outcomes
- `.clarc/skills/` directory scanning at session start
- Local skill override — project skill wins over global if same name
- `skills-local` command — lists all project-local skills
- `team-sync` command — syncs instincts across team members
